Each figure is generated by a self-contained Python script (numpy + matplotlib + pandas only). PNGs are 200 dpi for screen; PDFs are vector for print. All scripts can be re-run from this directory in seconds.

```bash
cd figures/
for fig in fig*.py; do python "$fig"; done
```

## Style choices

- Font: Arial / Helvetica (sans-serif) for legibility at print sizes.
- Colour palette: Paul Tol's "bright" scheme (colour-blind friendly): `#4477AA` (blue), `#EE6677` (red), `#CCBB44` (yellow), `#AA3377` (purple), `#888888` (grey).
- Hex topology = blue throughout; square topology = red throughout. Consistent across Fig 4 and any future topology figure.
- Merkabit ideal value = red highlighted line (Fig 5) or black ✕ marker (Fig 4).
- Poisson reference line (F = 1) = grey dashed.
- Top and right axes spines suppressed (Tufte-style).
